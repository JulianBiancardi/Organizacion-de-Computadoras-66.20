# Compilación

Se provee un archivo Makefile. Para utilizarlo, dirigirse hasta la carpeta TP1 en su computadora. Los siguientes comandos están disponibles

* _make_: compila el ejecutable
* _make tests_: compila las pruebas
* _make clean_: elimina los ejecutables compilados

# Pruebas

El resultado de las pruebas se imprime por salida estándar. Las pruebas de velocidad pueden ser excesivas para ciertos procesadores aunque casi no consumen memoria. En caso de que se note que no terminan, modificar la constante `SAMPLE_SIZE` del archivo `speed_tests.c` a un número menor y recompilar.
